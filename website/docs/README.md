# Website Docs

Operational documentation for the Astro website.

## Launch Checklist

- Run `npm run check`.
- Run `npm run build`.
- Preview the production build with `npm run preview`.
- Check core routes: `/`, `/about`, `/articles`, `/lessons`, `/topics`.
- Check one article page and one lesson page.
- Verify `robots.txt`, sitemap, and RSS output.
- Review mobile layout for the home page, article index, article page, and topic page.
- Confirm draft content is not listed.

## Content Review Checklist

- Clear reader and job-to-be-done.
- Concrete takeaway near the top.
- Complete title, description, slug, date, pillar, and status.
- Internal links to relevant topic or article pages.
- Citations for factual claims that depend on external research.
- Mobile-readable headings, lists, and code blocks.
