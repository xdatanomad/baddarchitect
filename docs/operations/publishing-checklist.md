# Publishing Checklist

- The reader and job-to-be-done are clear.
- The first screen includes a concrete takeaway.
- Factual claims that need evidence include citations.
- The title, description, canonical slug, and internal links are ready.
- The article or lesson has complete MDX frontmatter in `website/src/content/`.
- `draft: true` is set if the piece should not appear in production lists.
- `npm run check` passes from `website/`.
- `npm run build` passes from `website/`.
